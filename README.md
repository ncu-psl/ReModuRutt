## 簡介
此專案為製作一個能讓使用者依照簡單的規則，進行各種不同轉換的工具。(包含文字轉換、程式轉換或特定pattern偵測等)
使用者可以自行依照規則，新增欲進行的轉換方式。
- 執行平台 : Visual Studio 2017
- 撰寫語言 : C#
- 類型 : visual studio 2017 plug-in
